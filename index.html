Eisenhower

Eisenhower is a work-in-progress read-only rubygem to help you work the Interstate API.

  • First thing to do is set an OAuth token â€“ how you get this, at the moment, is up to you.

    {% highlight ruby %}Eisenhower.oauth_token = '432n4u3h24v23'{% endhighlight %}

  • Now you can get all your roadmaps!

    {% highlight ruby %}Eisenhower::Roadmap.all{% endhighlight %}

  • You'll get back interconnected Eisenhower objects whenever possible.

    {% highlight ruby %}map = Eisenhower::Roadmap.all.first map = Eisenhower::Roadmap.find("234f23f23fc2fg") # map.roads [#] map.roads.first.title "This is a road" map.roads.first.staff [#] {% endhighlight %}

    The main objects currently implemented, with closely-matching attributes, are:

    1. Roadmaps
    2. Roads
    3. Road Updates
    4. Staff

There's much left #todo, so please feel free to fork it on github.
